## Purpose

This section is a quick reference area for tools, commands, syntax, and common usage patterns.

Use this section when you already know what you want to do and need the command or tool syntax quickly.

Methodology and explanation should live in the main workflow sections.

This section should focus on:

- tool syntax
- common flags
- quick examples
- useful options
- common errors
- links back to methodology notes
