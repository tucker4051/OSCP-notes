## Purpose
Post-enumeration Swiss army knife for SMB, WinRM, LDAP, MSSQL, etc.

Used for:
- credential validation
- password spraying
- user enumeration
- SAM dumping
- command execution

## Common Commands

### Check SMB access
`crackmapexec smb <target>`

### Login with credentials
`crackmapexec smb <target> -u user -p password

### Check local admin access
`crackmapexec smb <target> -u user -p password --local-auth

### Password spraying
`crackmapexec smb <target> -u users.txt -p 'Password123'

### Dump SAM hashes
`crackmapexec smb <target> -u user -p password --sam

### Enumerate users
`crackmapexec smb <target> -u user -p password --users

### Enumerate shares
`crackmapexec smb <target> -u user -p password --shares

### Execute command
`crackmapexec smb <target> -u user -p password -x "whoami"

### WinRM login check
`crackmapexec winrm <target> -u user -p password
