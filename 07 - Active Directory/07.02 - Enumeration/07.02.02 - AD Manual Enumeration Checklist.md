## 1. Prefer RDP for Enumeration

Use RDP when possible:

```bash
xfreerdp /u:<user> /d:<domain> /v:<target_ip>
```

Reason:

```text
RDP avoids many Kerberos double-hop issues that can affect WinRM-based enumeration.
```

## 3. Build / Import LDAPSearch Function

Use this when PowerView is unavailable or to understand raw LDAP enumeration.

```powershell
function LDAPSearch {
    param (
        [string]$LDAPQuery
    )

    $PDC = [System.DirectoryServices.ActiveDirectory.Domain]::GetCurrentDomain().PdcRoleOwner.Name
    $DistinguishedName = ([adsi]'').distinguishedName
    $DirectoryEntry = New-Object System.DirectoryServices.DirectoryEntry("LDAP://$PDC/$DistinguishedName")
    $DirectorySearcher = New-Object System.DirectoryServices.DirectorySearcher($DirectoryEntry, $LDAPQuery)

    return $DirectorySearcher.FindAll()
}
```

Import:

```powershell
Import-Module .\function.ps1
```

Useful queries:

```powershell
LDAPSearch -LDAPQuery "(samAccountType=805306368)"
LDAPSearch -LDAPQuery "(objectCategory=group)"
LDAPSearch -LDAPQuery "(objectCategory=computer)"
LDAPSearch -LDAPQuery "(name=<username>)"
LDAPSearch -LDAPQuery "(&(objectCategory=group)(cn=<group name>))"
```

## 8. Unroll Nested Groups

If a group contains another group, enumerate the nested group too.

```powershell

```

Repeat until only users remain.

Record:

```text

indirect users
```

## 13. Enumerate Object Permissions / ACLs

Check interesting users, groups, and computers.

```powershell
Get-ObjectAcl -Identity "<object>"
```

Filter for dangerous rights:

```powershell
Get-ObjectAcl -Identity "<object>" |
? {$_.ActiveDirectoryRights -eq "GenericAll"} |
select SecurityIdentifier,ActiveDirectoryRights
```

Also check for:

```text
GenericWrite
WriteDACL
WriteOwner
AllExtendedRights
ForceChangePassword
Self-Membership
```

Convert SIDs:

```powershell
Convert-SidToName <SID>
```

Multiple SIDs:

```powershell
"<SID1>","<SID2>","<SID3>" | Convert-SidToName
```

High-value findings:

```text
normal user has GenericAll over group
normal user can reset another user's password
normal user can modify group membership
normal user can edit ACLs
```

## 14. Validate Group Control Carefully

If authorised and reversible, test group control.

Add user:

```powershell
net group "<group>" <user> /add /domain
```

Verify:

```powershell
Get-NetGroup "<group>" | select member
```

Clean up:

```powershell
net group "<group>" <user> /del /domain
```

Verify cleanup:

```powershell
Get-NetGroup "<group>" | select member
```

## 17. Hunt Custom Shares for Credentials

Browse shares:

```powershell
ls \\<host>\<share>
ls \\<host>\<share> -Recurse
```

Search useful file types:

```powershell
ls \\<host>\<share> -Recurse -Include *.txt,*.xml,*.config,*.ini
```

Search for credential keywords:

```powershell
Select-String -Path "\\<host>\<share>\*" -Pattern "password","passwd","pwd","credential","secret" -Recurse
```

Look for:

```text
passwords
onboarding emails
config files
backup files
scripts
web.config
unattend.xml
KeePass databases
RDP files
SSH keys
connection strings
```

## 18. Build Attack Path Notes

For every finding, record:

```text
finding
source command
source host
source file path
affected user/group/computer
privilege implication
possible next step
cleanup required
```

Example:

```text
stephanie has local admin on CLIENT74.
PsLoggedOn shows jeffadmin logged on to CLIENT74.
Possible path: access CLIENT74 and investigate credential material.
```

# Quick Priority Order

```text
1. Confirm user, host, domain, DNS, logon server
2. Enumerate users
3. Enumerate groups
4. Check privileged groups
5. Unroll nested groups
6. Enumerate computers and OS versions
7. Find local admin access
8. Find logged-on users
9. Enumerate SPNs and service accounts
10. Check ACLs on interesting objects
11. Enumerate shares
12. Search SYSVOL and custom shares
13. Record credentials and password patterns
14. Connect findings into attack paths
15. Repeat after every new credential or host
```

# Key Takeaway

AD enumeration is a loop:

```text
Enumerate
↓
Find relationship
↓
Validate carefully
↓
Document
↓
Pivot or gain new context
↓
Enumerate again
```
