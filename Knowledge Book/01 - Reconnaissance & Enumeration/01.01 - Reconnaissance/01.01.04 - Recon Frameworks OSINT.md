## Overview

Reconnaissance frameworks automate OSINT and web enumeration tasks by aggregating data from multiple sources.

Typical capabilities:

- subdomain discovery
- DNS enumeration
- email harvesting
- infrastructure mapping
- SSL certificate analysis
- historical URL discovery
- employee information gathering
- attack surface mapping

Useful during **early recon phase** to build target scope quickly.

---

## Tool Comparison

| Tool | Primary Use | Strength |
|------|-------------|---------|
| FinalRecon | Web recon automation | fast multi-module scanning |
| Recon-ng | modular recon framework | highly extensible |
| theHarvester | email & subdomain discovery | excellent OSINT collection |
| SpiderFoot | automated OSINT collection | integrates many APIs |
| OSINT Framework | tool directory | research reference |

# FinalRecon

## Overview

Python-based reconnaissance tool combining multiple recon techniques.

Collects:

- headers
- SSL info
- DNS records
- subdomains
- directories
- historical URLs
- internal/external links

Good first-pass recon tool.

---

## Installation

```bash
git clone https://github.com/thewhiteh4t/FinalRecon.git

cd FinalRecon

pip3 install -r requirements.txt

chmod +x finalrecon.py
```

---

## Usage

```bash
./finalrecon.py --help
```

---

## Common Commands

### full scan

```bash
./finalrecon.py --full --url http://target.com
```

---

### headers + whois

```bash
./finalrecon.py --headers --whois --url http://target.com
```

---

### ssl certificate info

```bash
./finalrecon.py --sslinfo --url http://target.com
```

---

### dns enumeration

```bash
./finalrecon.py --dns --url http://target.com
```

---

### subdomain enumeration

```bash
./finalrecon.py --sub --url http://target.com
```

---

### directory discovery

```bash
./finalrecon.py --dir --url http://target.com
```

---

### crawler

```bash
./finalrecon.py --crawl --url http://target.com
```

---

### wayback urls

```bash
./finalrecon.py --wayback --url http://target.com
```

---

## Modules

| Option | Description |
|--------|------------|
| --headers | analyse HTTP headers |
| --sslinfo | SSL certificate details |
| --whois | domain registration info |
| --crawl | crawl site links |
| --dns | DNS records |
| --sub | subdomains |
| --dir | directory brute force |
| --wayback | historical URLs |
| --ps | port scan |
| --full | full scan |

# Recon-ng

## Overview

Modular reconnaissance framework similar to Metasploit.

Supports:

- DNS enumeration
- subdomain discovery
- API integrations
- credential gathering
- infrastructure mapping

---

## Start framework

```bash
recon-ng
```

---

## Common workflow

```bash
marketplace search

marketplace install all

modules search

modules load recon/domains-hosts/brute_hosts

run
```
