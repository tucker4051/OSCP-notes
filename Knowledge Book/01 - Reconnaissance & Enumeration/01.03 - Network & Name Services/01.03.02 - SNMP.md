## Overview
SNMP (Simple Network Management Protocol) is commonly exposed on UDP **161** and is often misconfigured with weak or default community strings (e.g. `public`, `private`).

SNMP can leak:
- Usernames
- Installed software
- Running processes
- Network interfaces
- System information
- Credentials (rare but valuable)

---

## Quick Workflow

1. Check if SNMP is exposed
2. Identify community string
3. Enumerate OIDs for useful data

---

## Default Ports
| Port | Protocol | Description |
|------|----------|-------------|
| 161 | UDP | SNMP queries |
| 162 | UDP | SNMP traps |

---

## snmpwalk

### Purpose
Query SNMP OIDs to retrieve structured information.

### Syntax
```bash
snmpwalk -v2c -c public {target IP}
```

### Example
```bash
snmpwalk -v2c -c public 10.10.10.10
```

### Useful OID trees
| OID | Description |
|-----|------------|
| 1.3.6.1.2.1.1 | System info |
| 1.3.6.1.2.1.25.4.2.1.2 | Running processes |
| 1.3.6.1.2.1.25.6.3.1.2 | Installed software |
| 1.3.6.1.4.1 | Vendor specific |

---

## onesixtyone

### Purpose
Brute-force SNMP community strings.

Community strings act like passwords.

### Syntax
```bash
onesixtyone -c /opt/useful/seclists/Discovery/SNMP/snmp.txt {target IP}
```

### Example
```bash
onesixtyone -c /opt/useful/seclists/Discovery/SNMP/snmp.txt 10.10.10.10
```

### Common community strings
- public
- private
- manager
- admin

---

## braa

### Purpose
Fast SNMP enumeration / brute-force OIDs once community string is known.

### Syntax
```bash
braa <community string>@<IP>:.1.3.6.*
```

### Example
```bash
braa public@10.10.10.10:.1.3.6.*
```

## Detection Clues During Enumeration

SNMP may reveal:
- usernames for password spraying
- running services not visible via nmap
- internal hostnames
- installed software versions
- backup scripts or cron jobs
- network interface details

## Cheatsheet

```bash
# brute-force community strings
onesixtyone -c /opt/useful/seclists/Discovery/SNMP/snmp.txt {IP}

# enumerate with known string
snmpwalk -v2c -c public {IP}

# fast OID brute-force
braa public@{IP}:.1.3.6.*
```
