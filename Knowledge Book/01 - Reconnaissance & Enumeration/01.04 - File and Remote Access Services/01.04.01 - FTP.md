## Overview

FTP (File Transfer Protocol) allows file upload/download between systems.

Common weaknesses:

- anonymous login enabled
- weak credentials
- writable directories
- sensitive files exposed
- misconfigured permissions
- cleartext authentication
- TLS misconfiguration

---

## Default Ports

| Port | Protocol | Service |
|------|----------|--------|
| 21 | TCP | FTP |
| 20 | TCP | FTP Data |
| 69 | UDP | TFTP |

---

## Quick Workflow

1. Identify FTP service
2. Attempt anonymous login
3. Enumerate accessible files
4. download interesting files
5. check write permissions
6. test TLS configuration

---

## Anonymous Login

```bash
ftp {IP}
```

Login:

```
Name: anonymous
Password: anonymous
```

Basic enumeration:

```bash
ls
pwd
status
```

---

## Download Files

Single file:

```bash
get file.txt
```

Example:

```bash
get Important\ Notes.txt
```

Download all files:

```bash
wget -m --no-passive ftp://anonymous:anonymous@{IP}
```

---

## Upload Files

If write permissions enabled:

```bash
put test.txt
```

Example:

```bash
put shell.php
```

---

## Useful FTP Commands

| Command | Description |
|--------|-------------|
| ls | list files |
| cd | change directory |
| pwd | print working directory |
| get file | download file |
| put file | upload file |
| status | show connection status |
| binary | set binary transfer mode |
| ascii | set ascii transfer mode |
| debug | show debug info |
| trace | show packet trace |
| quit | exit ftp |

---

## Banner Grabbing

```bash
nc -nv {IP} 21
```

or

```bash
telnet {IP} 21
```

May reveal:
- FTP version
- hostname
- OS details

---

## Nmap Enumeration

```bash
sudo nmap -sV -p21 -sC -A {IP}
```

Useful NSE scripts:

| Script | Purpose |
|--------|--------|
| ftp-anon | checks anonymous login |
| ftp-syst | shows system type |
| ftp-bounce | checks bounce attack |
| ftp-proftpd-backdoor | detects backdoor |
| ftp-vsftpd-backdoor | detects vsftpd vuln |

---

## FTPS (FTP over TLS)

If TLS enabled:

```bash
openssl s_client -connect {IP}:21 -starttls ftp
```

Reveals:
- certificate details
- domain names
- organisation info

# TFTP Enumeration

## Overview

TFTP (Trivial File Transfer Protocol):

- uses UDP
- no authentication
- commonly used for configs/backups
- often misconfigured

---

## Default Port

| Port | Protocol | Service |
|------|----------|--------|
| 69 | UDP | TFTP |

---

## Connect to TFTP

```bash
tftp {IP}
```

---

## Useful TFTP Commands

| Command | Description |
|--------|-------------|
| connect | set remote host |
| get file | download file |
| put file | upload file |
| status | show transfer status |
| verbose | enable verbose output |
| quit | exit |

---

## Example Interaction

```bash
tftp {IP}

connect {IP}

get config.txt

put test.txt

quit
```

---

## High Value Files

Look for:

- config files
- backups
- credentials
- SSH keys
- scripts
- database exports

Common extensions:

- .conf
- .config
- .bak
- .backup
- .sql
- .xml
- .ini
