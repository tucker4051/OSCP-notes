## Purpose
Interact with SMB shares.

## Common Commands

### List shares
`smbclient -L //<target> -U user

### Connect to share
`smbclient //<target>/share -U user

### Anonymous access
`smbclient -L //<target> -U ""

### Download file
`get file.txt

### Upload file
`put shell.exe

## Notes
Useful for discovering credentials in shares.
