## Purpose
Interact with Windows RPC service.
Useful when SMB null sessions are allowed.

## Common Commands

### Connect anonymously
`rpcclient -U "" <target>

### Connect with credentials
`rpcclient -U user <target>

### Enumerate users
`enumdomusers

### Enumerate groups
`enumdomgroups

### Query group members
`querygroupmem <RID>

### Query user info
`queryuser <RID>

### Get domain info
`querydominfo

### Get password policy
`getdompwinfo

## Notes
Often useful when SMB allows anonymous access.
