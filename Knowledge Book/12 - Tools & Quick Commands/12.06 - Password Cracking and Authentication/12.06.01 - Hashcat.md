
## Basic Syntax

```bash
hashcat -a <attack_mode> -m <hash_mode> <hash> [wordlist/mask/rules]
```

Example:

```bash
hashcat -a 0 -m 0 hash.txt rockyou.txt
```

---

# Identify Hash Type

```bash
hashid hash.txt
```

Show hashcat mode:

```bash
hashid -m hash.txt
```

List supported hash modes:

```bash
hashcat --help
```

# Attack Modes

| Mode | Description |
|------|-------------|
| 0 | dictionary |
| 1 | combinator |
| 3 | mask (bruteforce pattern) |
| 6 | hybrid wordlist + mask |
| 7 | hybrid mask + wordlist |

---

# Dictionary Attack

```bash
hashcat -a 0 -m <mode> hashes.txt wordlist.txt
```

Example:

```bash
hashcat -a 0 -m 1000 ntlm.txt rockyou.txt
```

---

# Dictionary + Rules

Apply transformations:

```bash
hashcat -a 0 -m <mode> hashes.txt wordlist.txt -r rules.rule
```

Example:

```bash
hashcat -a 0 -m 0 hash.txt rockyou.txt -r /usr/share/hashcat/rules/best64.rule
```

---

# Mask Attack

Brute force with structure:

```bash
hashcat -a 3 -m <mode> hash.txt '?mask'
```

Example:

```bash
hashcat -a 3 -m 1000 hash.txt '?u?l?l?l?l?d?d'
```

---

# Mask Charset Reference

| Symbol | Meaning |
|--------|---------|
| ?l | lowercase |
| ?u | uppercase |
| ?d | digits |
| ?s | symbols |
| ?a | all printable |
| ?h | hex lowercase |
| ?H | hex uppercase |
| ?b | all bytes |

---

# Custom Charset

Define custom charset:

```bash
hashcat -a 3 -m 0 hash.txt -1 ?l?d '?1?1?1?1?1?1'
```

---

# Hybrid Attacks

Append mask to wordlist:

```bash
hashcat -a 6 -m 0 hash.txt wordlist.txt '?d?d'
```

Prepend mask:

```bash
hashcat -a 7 -m 0 hash.txt '?d?d' wordlist.txt
```
