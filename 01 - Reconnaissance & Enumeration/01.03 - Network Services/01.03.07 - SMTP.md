## Overview

SMTP (Simple Mail Transfer Protocol) is used for sending email between servers.

Misconfigured SMTP services may allow:
- User enumeration
- Open relay abuse
- Internal domain discovery
- Information disclosure via banners
- Credential attacks (password spraying)

Mail flow:

Client (MUA) → Submission Agent (MSA) → Mail Transfer Agent (MTA) → Mail Delivery Agent (MDA) → Mailbox (POP3/IMAP)

SMTP response codes reference:
https://serversmtp.com/smtp-error/

| Port | Protocol | Service |
|------|----------|--------|
| 25 | TCP | SMTP |
| 465 | TCP | SMTPS |
| 587 | TCP | SMTP Submission |
| 110 | TCP | POP3 |
| 143 | TCP | IMAP |

## Quick Workflow

1. Identify SMTP service
2. Enumerate supported commands
3. Attempt user enumeration
4. Check for open relay misconfiguration
5. Identify internal domains

---

## Banner Grabbing

```bash
nc -nv {IP} 25
```

or

```bash
telnet {IP} 25
```

Example output may reveal:
- mail server software
- internal domain name
- hostname

---

## HELO / EHLO Enumeration

EHLO often reveals supported SMTP features.

```bash
telnet {IP} 25

EHLO test
```

Example:

```bash
telnet 10.10.10.10 25

EHLO mail.example.com
```

Useful information:
- supported authentication methods
- server capabilities
- internal domain names

---

## VRFY User Enumeration

VRFY attempts to confirm whether a user exists on the mail server.

```bash
VRFY root
VRFY admin
VRFY test
```

Possible responses:

| Code | Meaning |
|------|--------|
| 250 | User exists |
| 252 | Cannot verify user (may still exist) |
| 550 | User does not exist |

⚠️ Many servers disable VRFY.

## Proxy Pivoting to SMTP

If working through a web proxy:

```http
CONNECT {IP}:25 HTTP/1.0
```

Example:

```http
CONNECT 10.129.14.128:25 HTTP/1.0
```

Useful when direct SMTP access is restricted.

## Nmap SMTP Enumeration

### Enumerate supported commands

```bash
sudo nmap {IP} -p25 --script smtp-commands
```

Uses EHLO to identify available commands.

---

### Check for open relay

```bash
sudo nmap {IP} -p25 --script smtp-open-relay
```

Tests for open relay misconfiguration.

---

## User Enumeration Tool

### smtp-user-enum

Supports VRFY, EXPN, RCPT attacks.

```bash
smtp-user-enum -M VRFY -U users.txt -t {IP}
```

Example:

```bash
smtp-user-enum -M VRFY -U /usr/share/seclists/Usernames/top-usernames-shortlist.txt -t 10.10.10.10
```

## Cheatsheet

```bash
# banner grab
nc -nv {IP} 25
telnet {IP} 25

# enumerate capabilities
EHLO test

# enumerate users
VRFY root

# nmap enumeration
sudo nmap {IP} -p25 --script smtp-commands
sudo nmap {IP} -p25 --script smtp-open-relay

# automated user enumeration
smtp-user-enum -M VRFY -U users.txt -t {IP}
```
